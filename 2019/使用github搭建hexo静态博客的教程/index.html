1. 安装 Hexo前，需要安装：

  • Node.js (建议使用 Node.js 10.0 及以上版本)
    1) Hexo是基于nodeJS环境的静态博客，是需要使用JavaScript的包管理工具npm安装。
    2) hexo-cli，负责创建 hexo 工程。
    3) hexo 模块，负责真正的项目管理工作（生成静态页面、预览等）。

  • Git

2. 安装hexo命令：

2.1 先在桌面创建一个文件夹（blog），用来存放所有静态博客的文件。
2.2 运用cd命令进入此文件夹，并使用npm安装Hexo：
$ cd blog
$ npm install -g hexo-cli
2.3 安装完成后，输入初始化命令，新建一个网站：
$ hexo init
2.4 输入初始化命令出现以下错误，则需要配置git的环境变量，若初始化命令运行成功则不需要设置：

2.5 设置环境变量的系统变量，新建变量（GIT_HOME）和值（C:\Program Files\Git）：

2.6 以及Path的值后追加（%GIT_HOME%\bin）：

2.7 设置完成后再运行初始化命令。
2.8 初始化完成后，blog文件夹会新增以下文件：

  • node_modules：是依赖包
  • public：存放的是生成的页面
  • scaffolds：命令生成文章等的模板
  • source：用命令创建的各种文章
  • themes：主题
  • _config.yml：整个博客的配置
  • db.json：source解析所得到的
  • package.json：项目所需模块项目的配置信息

4. 搭建github：

4.1 登录github官网，没有账号需要创建账号，有账号的新建repository：


注意：注册时填写的邮箱需要接收验证码，确保能正常使用。

4.2 设置资料库的信息，点击Create repository创建：


注意：仓库名字必须是：username.github.io，其中username是用户名。

以下操作均在Git Bash程序上执行：
4.3 打开Git Bash程序，并设置github账号信息：
$ git config --global user.name "xxx"// 你的github用户名，非昵称
$ git config --global user.email  "xxx@hotmail.com"// 填写你的github注册邮箱
4.4 配置SSH Key，使用ssh key连接：
4.4.1 输入以下命令检查本机已存在的ssh密码：
$ cd ~/.ssh
4.4.2 若出现 No such file or directory说明是第一次使用Git，输入以下命令后，连续按三次Enter按钮：
$ ssh-keygen -t RSA -C "邮件地址"
4.4.3 根据以下回显，打开.ssh的文件夹后，会出现两个以id_rsa命名的文件：

4.4.4 使用记事本打开id_rsa.pub文件，并复制文件内容，打开github主页，进入个人设置的 SSH and GPG keys，点击New SSH key，粘贴到key下的框内：

4.4.5 通过ssh连接GitHub，在Git Bash输入：
$ ssh -T git@github.com # 注意邮箱地址不用改

如果提示Are you sure you want to continue connecting (yes/no)?，输入yes后出现：
Hi GGingyii! You’ve successfully authenticated, but GitHub does not provide shell access.
则说明SSH已配置成功。

5. 使用hexo写博文：

5.1 新建一篇文章：
$ hexo new "使用github搭建hexo静态博客的教程"
5.2 在GitHub网页的资源库中，Hexo 会创建一个以标题为名字的目录，并在目录中放置一个 index.md 文件；在系统中会在_posts文件夹下创建一个以标题为名字的目录和md 文件：

5.3 在md文件内输入博文的内容，title为博文标题，在灰色框下输入博文内容：

5.4 在与md文件相同名字的目录下存放博文需要的照片。
5.5 将md文件生成静态文件：
$ hexo generate 或 $ hexo g
5.6 启动服务器：
$ hexo server
5.7 启动服务器。默认情况下，访问网址为： http://localhost:4000/
5.8 将静态文件上传到GitHub：
$ hexo d

如果出现”ERROR Deployer not found: git”错误，则执行以下命令后，再执行$ hexo d：

$ npm install hexo-deployer-git --save

6. 验收成果：

6.1 在浏览器上输入https://username.github.io/ 即可查看刚搭建的博客。
6.2 或输入http://localhost:4000/ 查看。
